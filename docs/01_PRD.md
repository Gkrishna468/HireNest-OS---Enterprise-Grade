# Product Requirements Document (PRD)

## Vision
HireNestOS is an AI-native staffing platform that serves as the single source of truth for intelligent candidate matching and vendor pipeline management, ensuring seamless collaboration and eliminating data drift in the recruitment lifecycle.

## Users
1. **Recruiters / Global Admins**: Manage requirements, oversee vendor performance, and administrate the global candidate pool.
2. **Vendors (Agencies)**: Receive open requirements, submit candidates efficiently, and track submission statuses.
3. **Clients (Hiring Managers)**: View matched candidates, advance pipeline stages, and provide feedback.

## Problems Solved
- Fragmented communication across multiple external vendors, internal recruiters, and hiring managers.
- Duplicate candidate submissions and ownership disputes.
- Inconsistent tracking of candidate stages and data drift.

## Core Features
1. **Recruiter Workspace**
2. **Vendor Workspace**
3. **Client Workspace**
4. **Candidate Ownership Vault**
5. **Submission Engine**
6. **AI Matching Engine**
7. **Event Ledger**
8. **Email Intelligence Layer**
9. **Multi-tenant Architecture**

## MVP Scope
Multi-role authentication, requirement distribution, vendor AI resume parsing and submissions, duplicate detection, client candidate review, and event logging.

## Success Metrics
- Reduced time-to-hire.
- Number of duplicate submissions caught to prevent vendor collisions.
- High match score accuracy leading to successful placements.
