# Domain Glossary

To keep terminology consistent across AI-generated code and documentation, the following definitions are strictly enforced:

| Term | Meaning |
|---|---|
| **Candidate** | A person registered in HireNest (stored in `candidatePool`). |
| **Vendor** | A third-party staffing partner or agency. |
| **Client** | A hiring company or department that creates requirements. |
| **Submission** | A candidate actively submitted to a specific requirement. |
| **Requirement** | A job requisition or open position. |
| **Ownership** | The entity (organization/user) authorized to manage a candidate or requirement. |
| **Deal Room** | A collaboration workspace for recruiters, clients, and vendors to discuss submissions. |
| **Match Matrix** | The global candidate–requirement semantic scoring index. |
